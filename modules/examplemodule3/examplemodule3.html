This module is in a separate view from "view_initial"

It can be called by calling the view that contains it using the following Javascript:

setView(*name_of_view*)

Keep in mind that you are calling an ARRAY and not a STRING; therefore, do not wrap the view name in quotes.

It is also worth noting that you should have noticed the page did not refresh or navigate to load this additional content.
Everything that is called with setView happens using XHR functionality. This allows you to create seemingly one-page websites or web apps.
