ADDING MODULES

[1] THE MODULE **MUST** BE OF .HTML FORMAT
[2] THE MODULE **MUST** BE INSIDE OF A SUBDIRECTORY WITH THE EXACT SAME NAME AS THE .HTML MOD FILE
[3] THE MODULE SUBDIRECTORY OF THE SAME NAME **MUST** BE WITHIN THE "MODULES" DIRECTORY

YOUR MODULE NEEDS TO BE IN THE FORMAT SHOWN HERE.
USE A TAG THAT HAS THE EXACT SAME NAME AS THE MODULE SUBDIRECTORY AND MODULE .HTML FILE
YOU MUST INCLUDE SCRIPT TAGS EVEN IF YOU ARE NOT INCLUDING ANY JAVASCRIPT

YOUR MODULE SHOULD HAVE AT LEAST ONE TOP LEVEL NESTED TAG

AN ID IS NOT REQUIRED ON THE TOP LEVEL NESTED TAG

IN SOME CASES YOU MAY SEE A RANDOM OPENING OR CLOSING TAG SIGN ON THE WEBPAGE. THIS CAN EASILY BE CORRECTED BY ADDING A TRAILING SPACE AFTER THE CLOSING TOP LEVEL NESTED TAG
